# C-Typing Tutor FastTyping

Um treinador de digitação via terminal focado em memória muscular, escrito em C, rodando no próprio terminal.

## Funcionalidades Atuais
- Filtra palavras do dicionário do Linux (`/usr/share/dict/words`).
- Gera palavras baseadas apenas nas teclas desbloqueadas (status.txt).
- Seleção aleatória e eficiente de palavras.
